# Heart Disease Prediction 🫀

## Overview
A Machine Learning project to predict whether a person has heart disease or not based on medical parameters.

## Dataset
- 303 records, 14 features
- Source: UCI Heart Disease Dataset

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, Seaborn, Matplotlib

## Models Compared
| Model | Test Accuracy |
|-------|--------------|
| Logistic Regression | 78.69% |
| Random Forest | 77.05% |
| SVM | 80.33% |
| KNN | 81.97% 🏆 |
| Decision Tree | 77.05% |

## Best Model
**KNN** with **81.97%** test accuracy
